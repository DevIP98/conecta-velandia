# ConectaVelandia – Guía de Uso

Aplicación educativa sobre redes domésticas para las familias de la vereda Velandia.  
**Proyecto de Grado Fase 4 · UNAD ·**

---

## ▶️ Cómo abrir la aplicación

1. Abre la carpeta del proyecto.
2. Haz doble clic en el archivo **`iniciar.bat`** (o sigue el paso manual abajo).
3. Espera unos segundos y abre tu navegador en: **http://localhost:8080**

### Si no tienes el archivo .bat (manual):
1. Abre una ventana de **Símbolo del sistema** (CMD) en la carpeta del proyecto.
2. Escribe este comando y presiona Enter:
   ```
   python -m http.server 8080
   ```
3. Abre tu navegador y ve a: **http://localhost:8080**

---

## 📚 Contenido de los 5 Módulos

1. 🌐 **Conceptos Básicos** – ¿Qué es una red? Tipos y dispositivos.
2. ⚙️ **Configuración del Router** – Cómo acceder y configurar el router doméstico.
3. 📶 **Optimización Wi-Fi** – Mejorar la señal y cobertura en el hogar.
4. 🔧 **Solución de Problemas** – Diagnóstico y pasos para resolver fallas.
5. 🔐 **Seguridad Informática** – Contraseñas, cifrado y protección de la red.

---

## ℹ️ Información técnica básica

- Desarrollado en: **HTML, CSS y JavaScript**
- Base de datos local: **SQLite** (sin necesidad de servidor externo)
- Funciona en: **Navegador web y dispositivos Android**
- Modo sin conexión: **Sí** (Service Worker)
